= AutoSupport の配信方法を設定するにはどうすればよいですか？
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AutoSupport 配信方法の設定タスクにアクセスするには、[Support][Support Center]メニューに移動し、AutoSupport タブをクリックします。

サポートされているプロトコルは、HTTPS、HTTP、およびSMTPです。
