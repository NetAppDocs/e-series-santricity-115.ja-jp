= ボリュームを非同期でミラーリング
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
非同期ミラーリング機能を使用するとボリュームの物理コピーを2つ作成できるため、データの可用性を高め、ダウンタイムを最小限に抑えることができます。すべての非同期ミラーリング関係には、プライマリロールのボリュームが1つ、セカンダリロールのボリュームが1つ（別のストレージアレイ上）必要です。
